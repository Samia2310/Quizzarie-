This is my project for CSE341: Microprocessor, a two-player quiz game named "Quizzarie", developed in 8086 Assembly language. It features user authentication, 50:50 lifelines, turn-based gameplay, and real-time scoring, creating a simple yet engaging competitive assembly-based quiz experience.
